Custom Checkbox

Unlike default input based checkbox custom checkbox is not input based and doesn't have value. Rather it has aria value "on" and "off". Custom checkbox can be styled and interacted with. Same logic can be be used to create visual switches as well. It starts with default aria-valuetext as "off".
